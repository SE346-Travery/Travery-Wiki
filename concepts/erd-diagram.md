# Concept: ERD Diagram

## Definition
Entity Relationship Diagrams used to model the [[Travery]] database schema.

## Levels of Detail
- **Conceptual ERD**: High-level entities and business relationships.
- **Physical ERD**: Detailed database implementation including data types, keys, and constraints.
- **Latest ERD**: The most current version of the system schema.
