# Role: Khách hàng (Customer / Tourist)

## Description
The primary user of the [[travery]] application who consumes travel services.

## Responsibilities
- Searching for tours, buses, and hotels.
- Providing accurate guest/member information.
- Completing 100% payments within specified hold times.
- Following journey updates and reporting incidents.
- Evaluating services after completion (1-5 star ratings).
